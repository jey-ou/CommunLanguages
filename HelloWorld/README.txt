Hello World is een van de eerste oefeningen die vaak gebruikt worden bij het aanleren van een nieuwe taal.
Het lijkt een eenvoudige oefening maar bevat reeds heel wat onderdelen die je onder de knie moet hebben om te programmeren.

De herkomst van vele populaire talen is te vinden in c. Een kenmerk hiervan is de main functie (java: main methode).
In c, c++, java krijgt de main functie een belangrijke taak weggelegd. Indien aanwezig wordt deze functie eerst uitgevoerd als het programma wordt uitgevoerd.
Voor programma's die gecompileerd worden, wordt via de main functie alle functionaliteiten opgestart. Indien deze functie of methode niet voorkomt, kan het programma een bibliotheek functie zijn die gebruikt wordt in andere programma's.
Je ziet de main functie (methode)ook vaak gebruikt worden om testen mee uit te voeren.
